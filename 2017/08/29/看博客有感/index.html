看博客有感

发表于 2017-08-29 | 分类于 program

“「什么都知道一点，但都不甚了解」是最可怕的”

在博客上看到了这句话，感觉完美的诠释了我的生活，一直半吊子可能还会继续半吊子的一生。

一直沉迷在小说、动漫、漫画、电视剧、游戏，反正只要不学习做啥都有趣。成绩在初中时是中上游，在高中时下游，上了一个二流大学，所幸的是选了一个喜欢的专业。然而自己依然是半吊子的过活。

小学的时候学过游泳、乒乓球、国画、书法、跳舞、奥数、葫芦丝，琴棋书画全都涉及过，却又样样不精；初中的时候喜欢漫画小说，写了很多矫情的东西，却也没能留住青涩的记忆；现在回忆起来，感觉高中过得最为黑暗，也最为苦涩，在班级倒数的我，经历了一系列的感情变化，性格也逐渐改变，变得封闭又开朗。

一路走来，我始终认为自己是极度幸运的，考上了自主招生，去了一所传说中的末流985，顺利的转入了喜欢的专业，遇到了很好的老师和同学，还有亲人们对我的支撑。我本该无所畏惧。

但是，我仍然是我，清楚地知道自己的劣根性。不断纵容自己，给自己定的deadline永远在往后拖延，安慰自己这并不是最后的时刻。痛恨极了这样的自己，却又无力改变。我的决心，永远只存活三天。

按理说，写了这篇有感，应该是要激励自己进取向上，但是类似的情绪出现了太多次，多到自己懒得说出这种激愤的话语，冷暖自知便是。
